## Overview

SportsGPT is a domain-specific AI application built for Section XI high school football — covering Nassau and Suffolk County schools on Long Island. The platform ingests five seasons of game data, box scores, and player statistics, and presents them through a fast, interactive UI built for fans, coaches, and recruiters.

## Key Features

- **Game Intelligence**: Every game has a dedicated page with quarter-by-quarter scoring, full player stat breakdowns (passing, rushing, receiving, kicking, defense), and AI-rewritten narrative highlights in a beat-reporter voice.
- **Player Profiles**: Season-by-season career stats with per-game logs, position filtering, and direct links between game box scores and player detail pages.
- **Interactive Maps**: A Suffolk County choropleth map colored by school identity — click any district to open a team profile without leaving the map.
- **Charts & Leaderboards**: Cross-team statistical comparisons — point differential leaders, scoring streaks, home vs. away records — filterable by season.
- **Photo Gallery**: A Blob-backed masonry photo gallery with lightbox, dark/light toggle, and View Transition API animations.

## Technologies Used

- **Next.js 16 (App Router)**: Server components, parallel routes, and Turbopack for a fast, composable UI.
- **Neon Postgres**: Serverless Postgres for all game, player, team, and box score data across five seasons.
- **Claude API (Haiku)**: AI highlight rewriter — transforms raw scraped text into polished game summaries, cached permanently per game.
- **Vercel Blob**: Stores all team photos, player headshots, school logos, and GeoJSON assets outside the repo.
- **Recharts + Google Maps**: Data visualization and district boundary rendering.

## Challenges and Learnings

The biggest challenge was data quality. Game and player records were scraped from Newsday and MaxPreps across five seasons, with inconsistent school slugs, missing Nassau County schools, and JSONB player stat blobs that needed to be reconciled against a relational player table. Building a robust ingest pipeline with fuzzy name matching and manual gap-filling was as much work as the UI itself.

Designing the "stay-on-page" UX pattern — where clicking a player, team, or game opens a detail panel rather than navigating away — required a parallel route architecture that is now the canonical template for all new GPT applications in the portfolio.

## Outcome

SportsGPT covers 1,100+ games across five seasons (2021–2026), 300+ player profiles, and every Section XI school. It serves as both a live product and the reference architecture for a growing suite of domain-specific AI applications built on the same stack.
