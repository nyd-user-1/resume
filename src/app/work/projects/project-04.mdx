## Overview

ChildcareGPT is a domain-specific AI application built for New York State families, caseworkers, and policy researchers. It aggregates 16,800 OCFS-regulated childcare facilities with county-level cost data from the DOL National Database of Childcare Prices (2008–2022), and wraps the NY State OCFS-6025 subsidy application in an AI-assisted guided form — making a notoriously complex government process navigable for the families who need it most.

## Key Features

- **Facility Explorer**: Browse and filter all 16,800 OCFS-regulated providers (Day Care Centers, Family Day Care, Group Family Day Care, School-Age) by county, program type, and status. Each facility has a dedicated profile with capacity breakdowns by age group (infant, toddler, preschool, school-age), licensing dates, and contact information.
- **County Cost Profiles**: County-level childcare cost data from the DOL database — median and 75th-percentile center and family care costs by age group, women's labor force participation, household income, and poverty rates — visualized as interactive charts across 15 years of data.
- **Interactive Choropleth Map**: NY State county map colored by childcare cost metrics, with click-through to county detail pages.
- **Eligibility Application**: A guided 5-tab implementation of the OCFS-6025 subsidy application form with a yellow dot progress system, section confirm/warn/collapse states, household composition table, and absent parent section.
- **AI Chat**: Context-aware assistant that answers questions about eligibility rules, facility types, and program requirements.

## Technologies Used

- **Next.js 16 (App Router)**: Server components, dynamic routes, API routes, and a shared `GeoDetailPage` layout pattern inherited from the NYSgpt platform template.
- **Neon Postgres**: `facilities` table (16,800 rows, indexed on county, program type, ZIP, region) + `county_costs` table (930 rows, 2008–2022 by county FIPS).
- **data.ny.gov + DOL APIs**: Automated ingest scripts for OCFS facility data (two merged datasets) and DOL county cost CSVs.
- **Google Maps**: Facility location mapping and NY county boundary rendering.
- **Recharts**: Cost trend charts by age group and care type across the full 15-year dataset.

## Challenges and Learnings

The OCFS-6025 application form is one of the most complex government forms in New York State — 5 sections, conditional logic based on household composition, and eligibility rules that vary by income, family size, and reason for care. Modeling this as a stateful guided form (rather than a static PDF) required a careful state machine design with per-section validation, a global yellow-dot progress indicator, and collapsible sections that lock once confirmed.

The facility data required merging two separate `data.ny.gov` datasets — one for facility attributes and one for licensing dates — with careful deduplication before upsert into Neon.

## Outcome

ChildcareGPT demonstrates that government data — even when spread across multiple agencies, inconsistently formatted, and buried in legacy portals — can be transformed into a navigable, human-centered application. It is both a live product and a template for applying the NYSgpt platform architecture to social services and regulatory compliance domains.
